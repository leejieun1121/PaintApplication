# PaintApplication
그림판 어플리케이션

대학교 안드로이드 수업 때 만들었던 그림판 어플리케이션입니다. 

기존의 코드에서 

- clear 기능
- 펜 색 변경 기능
- 사이즈 변경 기능
- 그림 저장 기능
- 그림 불러오기 기능
- 도형 그리기 기능

등을 추가했습니다.
